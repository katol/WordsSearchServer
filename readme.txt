"WordsSearchServer" принимает текстовые сообщения по сети, сохраняет их в файл "output.txt" с указанием времени получения сообщения, а также проверяет сообщения на наличие заданных констант и в случае их обнаружения выводит такие сообщения в консоль.

Для использования "WordsSearchServer" 
1) Откройте командную строку
2) Перетащите мышкой в командную строку Server.bat
3) Введите через пробел константы, на наличие которых будет выполняться проверка сообщений
4) Нажмите клавишу "Enter"
5) Откройте новую командную строку
6) Перетащите мышкой в командную строку Client.bat
7) Адрес сервера (если запускаете на одном компьютере, то введите "localhost")
8) Нажмите клавишу "Enter"
9) Через клавишу "Enter" вводите сообщения, которые хотите отправить на сервера
10) Введите "exit" и нажмите клавишу "Enter", завершив тем самым работу с программой

Для быстрого тестирования приложения с константами "ab", "cde" и адресом сервера "localhost" 
1) Кликните дважды на Test.bat.
2) Через клавишу "Enter" вводите сообщения, которые хотите отправить на сервера
3) Введите "exit" и нажмите клавишу "Enter", завершив тем самым работу с программой

Основные исходные коды лежат в src/main/java
Модульные тесты лежат в src/test/java
При запуске приложения сообщения будут сохранены в "output.txt"
При запуске модульных тестов сообщения будут сохранены в "outputTest.txt"
